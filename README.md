# Advanc3D — Product Development & Prototyping Funnel

Engineering-reviewed additive manufacturing for medical device startups,
inventors, and engineering teams.

- Quote in 24 hours
- DFM feedback on every upload
- Functional prototypes in 3–7 business days
- NDA-ready intake · US-based
- HP MJF, SLA, FDM, and TPU in-house

## Stack

Next.js 14 · React 18 · TypeScript · Tailwind CSS · next/font (Manrope + Inter)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
